# ordbok

A webapp for looking up Swedish words when eg solving a crossword. The source of the dictionary is https://addons.mozilla.org/sv-SE/firefox/addon/g%C3%B6rans-hemmasnickrade-ordli/

App is availble at https://jabbors.github.io/ordbok/

## Project setup
```
yarn install
```
